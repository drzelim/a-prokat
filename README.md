### Открыть директорию `front` и установить зависимости:
> npm i

### Потом также установить зависимости в директории `server`:
> npm i

### Запустить `server`:
#### Указать .env переменные `BP_LOGIN` и `BP_PASSWORD`

> npm run start 
> 
или
> 
> npm run start:dev


### Запустить `front`:
> npm run serve
